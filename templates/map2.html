Fortune 1000: A Tilegram Story

A company's decision to locate in a particular state can bring revenue, jobs, and bragging rights. We are showing the relative size of states based on elements of the Fortune 1000. We've included a view for number of Fortune 1000 companies per state, aggregate revenue per state, and aggregate profit per state.
A tilegram creates a proportionate map - showing scale based on a given data set. Pitch Interactive (www.pitchinteractive.com) is a data visualization studio based in California. They created a portal allowing users to upload formatted csv files to create a tilegram that can be downloaded (as svg or topojson) and added to an html page using d3.
